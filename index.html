Welcome to my Twitch API How to Guide!

Twitch.tv is a platform that allows users to stream live videos for any audience. Twitch has come quite a long way, in the past most streams were of individual streamers playing video games, but now there are some video game tournaments that stream for hundreds of thousands of viewers. There are even IRL (in real life) streams which are similar to vlogs, except the streamer can interact with their users in real time (think Facebook live videos).

This how to guide will introduce you to the Twitch API and explore some functionalities that the API is capable of. The guide assumes that you have some fundamental HTML/CSS and Javascript knowledge.

This guide will go over the following:
  • What is an API?
  • Getting an API key
  • Making calls to the Twitch API using a request object
  • Full Sample Code
  • How to interpret the response from the server
  • More useful Twitch API calls
  • Conclusion
